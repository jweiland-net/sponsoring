============
Installation
============

Installation Type
=================

Composer
""""""""

You can install sponsoring with following shell command:

.. code-block:: bash

   composer req jweiland/sponsoring

Extensionmanager
""""""""""""""""

If you want to install `sponsoring` traditionally with Extensionmanager, follow these steps:

#. Visit ExtensionManager

#. Switch over to `Get Extensions`

#. Search for `sponsoring`

#. Install extension

DEV Version (GIT)
"""""""""""""""""

You can install the latest DEV Version with following GIT command:

.. code-block:: bash

   git clone https://github.com/jweiland-net/sponsoring.git
